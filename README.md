# 正方选课工具

专为多学校设计的现代化选课工具，提供美观易用的Web界面。

## 功能特点

- 学生信息管理 - 查看和刷新学生基本信息
- 课程信息浏览 - 查看可选课程和已选课程
- 智能选课系统 - 多种选课模式，自动抢课
- 课表查询 - 查看个人课程表
- 成绩查询 - 查询学期成绩和总体成绩
- 多学校支持 - 支持多个学校配置

## 技术栈

- Next.js 14 - React全栈框架
- TypeScript - 类型安全的JavaScript
- Tailwind CSS - 实用优先的CSS框架
- Framer Motion - 动画库

## 快速开始

### 环境要求

- Node.js 18+ 
- 现代浏览器

### 启动应用

```
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 访问地址

- 应用界面: http://localhost:3000
