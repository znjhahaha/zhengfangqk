# 多选抢课功能演示

## 功能演示步骤

### 1. 进入多选模式
1. 打开课程信息页面
2. 点击"多选模式"按钮
3. 界面会显示多选相关的操作按钮

### 2. 选择课程
1. 课程卡片变为可点击状态
2. 直接点击课程卡片来选择/取消选择
3. 选中的课程会显示绿色边框和勾选标记
4. 选中的课程标题会变为绿色
5. 右上角会显示已选择的课程数量

### 3. 批量操作
- **全选**: 点击"全选"按钮选择所有筛选结果
- **清空选择**: 点击"清空选择"按钮取消所有选择
- **批量抢课**: 点击"批量抢课"按钮开始批量抢课

### 4. 抢课过程
1. 系统会分批处理选中的课程
2. 每批最多3个并发请求
3. 实时显示每门课程的抢课结果
4. 完成后显示总结信息

## 界面变化

### 多选模式启用前
- 每个课程卡片右侧有"抢课"按钮
- 卡片不可点击选择
- 没有批量操作按钮

### 多选模式启用后
- 课程卡片变为可点击状态
- 左侧显示圆形选择指示器
- 选中的课程显示绿色边框和背景
- 右侧的"抢课"按钮隐藏
- 显示批量操作按钮组
- 显示多选统计卡片
- 显示操作提示卡片

## 技术特点

1. **智能分批**: 自动分批处理，避免服务器压力
2. **并发控制**: 限制并发数量，保证稳定性
3. **实时反馈**: 每门课程结果实时显示
4. **错误处理**: 单个失败不影响其他课程
5. **状态管理**: 完整的多选状态管理

## 使用建议

1. **选择数量**: 建议一次选择不超过10门课程
2. **网络环境**: 确保网络连接稳定
3. **Cookie状态**: 确保Cookie未过期
4. **时间安排**: 注意课程时间冲突

## 注意事项

- 多选模式只在"可选课程"标签页可用
- 批量抢课完成后会自动退出多选模式
- 抢课结果会实时显示，无需手动刷新
